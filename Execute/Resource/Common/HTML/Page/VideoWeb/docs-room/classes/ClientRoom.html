join

  • join(url: string, roomId: string, peerId: string, sig: string, appData: any): Promise<{ appData: any; peers: { appData: any; id: string }[] }>

  • 进入房间

    • 视频服务提供了创建房间、创建参与者（并生成签名信息）管理接口，这些管理接口应该由相应业务服务（如ACD排队服务）来调用然后将生成的房间ID、参与者ID、签名信息传给本SDK。
    • 成功进入房间，房间中的其他参与者将接收到【参与者进入房间事件({@link evt_peer_joined})】。

    Parameters

    • url: string

      webscoket服务器地址。必须以wss://开关（WebRTC只允许安全链接）

    • roomId: string

      房间ID

    • peerId: string

      参与者ID

    • sig: string

      签名信息

    • appData: any

      参与者自定义数据。视频服务将此数据转发给房间的其他参与者

    Returns Promise<{ appData: any; peers: { appData: any; id: string }[] }>

    • appData 房间自定义数据
    • peers 房间参与者列表
    • peers[x].id 参与者ID
    • peers[x].appData 参与者自定义数据
