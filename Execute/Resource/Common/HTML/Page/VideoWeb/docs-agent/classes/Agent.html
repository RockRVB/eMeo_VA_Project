getStatus

  • getStatus(): Promise<{ status: "ONLINE" | "AVAILABLE" | "SERVICE" | "AUX" | "ACW" | "SELECTED" }>

  • 获取坐席状态

    Returns Promise<{ status: "ONLINE" | "AVAILABLE" | "SERVICE" | "AUX" | "ACW" | "SELECTED" }>

    坐席的当前状态：

    • 'ONLINE' 在线但未就绪
    • 'AVAILABLE' 就绪，通过setStatus设置
    • 'INSERVICE' 服务中，ACD服务器在坐席接受通话后自动设置
    • 'AUX' 休息，通过setStatus设置
    • 'ACW' 文档处理中，ACD服务器在挂断通话后自动设置
